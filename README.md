# Heart Disease Prediction

## Introduction
This project uses Machine Learning to predict the likelihood of heart disease based on patient health parameters. A Streamlit web application was developed to provide real-time predictions.

## Objectives
- Predict heart disease risk using patient data
- Build a machine learning classification model
- Develop an interactive Streamlit application
- Support data-driven healthcare decisions

## Workflow
Data Collection → Data Preprocessing → Model Training → Model Evaluation → Streamlit Deployment

## Features
- User-friendly Streamlit interface
- Real-time prediction
- Machine Learning-based classification
- Input validation and preprocessing

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Model Information
- Algorithm: K-Nearest Neighbors (KNN)
- Data Scaling: StandardScaler
- Model Persistence: Joblib

## Project Files
- app.py
- KNN_heart.pkl
- scaler.pkl
- columns.pkl
- requirements.txt
